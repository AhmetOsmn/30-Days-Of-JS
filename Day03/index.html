3. Gün Notları

Kıyaslamalarda 2 tip değer vardır:

- Gerçek Değerler:
- 0 hariç bütün sayılar, (pozitif ve negatif) gerçektir.
- Boş string ("") hariç bütün string'ler gerçektir.
- Boolean olarak true bir gerçek değerdir.

- Sahte Değerler:
- 0
- 0n
- null
- undefined
- NaN
- Boolean olarak false
- Empty stringler: "", '', ``

Bir değişken tanımladığımızda ve bu değişkene bir değer ataması yapmadığımızda, bu değişkenin tipinin undefined olduğunu söylemiştik. Bu duruma ek olarak, yazdığımız bir fonksiyon geriye bir değer return etmiyor ise bu fonksiyon da undefined durumdadır.

Eşitlik kontolü yaparken iki farklı operatör kullanılabilir.

1. 'x == y' : x ve y değişkenlerinin değerlerini kıyaslar.
2. 'x === y' : x ve y değişkenlerinin değerlerini ve tiplerini kıyaslar.
Örnek:
console.log(3 == '3') --> true
console.log(3 === '3') --> false

console.log(0 == false) --> true
console.log(0 === false) --> false

console.log(0 == '') --> true
console.log(0 == ' ') --> true
console.log(0 === '') --> false
console.log(1 == true) --> true
console.log(1 === true) --> false
console.log(undefined == null) --> true
console.log(undefined === null) --> false
console.log(NaN == NaN) --> false
console.log(NaN === NaN) --> false

Window fonksiyonlarından prompt():

Ekrandan alınan bir değeri, bir değişkene atamak için kullanırız. İlk parametresi zorunludur. İkinci parametre placeholder görevini görür ve opsiyoneldir. Örnek kullanımı:
let number = prompt('Enter number','number goes here')
console.log(number)

Window fonksiyonlarından confirm():

Kullanıcıya bir onaylatma işlemi yapmamızı sağlar. İçerisine ekranda sorulması istenen mesajı yazarız.
confirm() fonksiyonu ekranda bir popup açar ve verilen mesaj ile birlikte ok ve cancel butonları gözükür. Burada ok -> true, cancel -> false şeklinde atama gerçekleştirilir. Örnek kullanımı:
const agree = confirm('Are you sure you like to delete? ')
console.log(agree)

Date objesi ile bazı örnek çalışmalar:

const now = new Date()
console.log(now) --> Mon Dec 12 2022 01:02:32 GMT+0300 (GMT+03:00)
console.log(now.getFullYear()) --> 2022
console.log(now.getMonth()) --> 11
console.log(now.getDate()) --> 12
console.log(now.getDay()) --> 1
console.log(now.getHours()) --> 1
console.log(now.getMinutes()) --> 2
console.log(now.getSeconds()) --> 32
