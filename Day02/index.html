2. Gün Notları

Veri tiplerini ikiye ayırabiliriz:

- Primitive tipler
- Primitive olmayan tipler (Obje Referansları)

Primitive Veri Tipleri

- Numbers
- Strings
- Booleans
- Null
- Undefined
- Symbol

Primitive Olmayan Veri Tipleri

- Objects
- Arrays

Primitive veri tipleri immutable (non-modifiable) (modifiye edilemez) veri tipleridir. Bir primitive tip oluşturulduktan sonra onu bir daha modifiye edemeyiz.

Primitive tipler içerisindeki verilere göre kıyaslanır (compared). Örnek olarak :

let hede = 'JS'
let bidi = 'JS'
let tede = 'JavaScript'
console.log(hede == bidi) --> true
console.log(hede == tede) --> false

Primitive olmayan veri türleri ise oluşturulduktan sonra değiştirilebilir, modifiye edilebilirler.

JS'te array'ler farklı tipte verileri içerebilirler.

Primitive olmayan veri tipleri aynı verileri içerseler bile kıyaslama yapıldıklarında eşit olarak kabul edilmezler. Örnek olarak:

let nums1 = [1,2,3]
let nums2 = [1,2,3]
console.log(nums1 == nums2) --> false

Primitive olmayan tipler Referance Type olarak da adlandırılabilirler. Referans tipler içerisindeki değerlere göre kıyaslanmazlar, referans oldukları adreslere göre kıyaslanırlar. Yani iki referans tipli değişkenin değerleri aynı olsa bile, farklı yerlere referans gösteriyorlarsa eşit değillerdir. Ancak iki değişken aynı yeri referans ediyor ise eşittirler. Örnek olarak:

let nums = [1,2,3]
let nums2 = nums
console.log(nums == nums2) --> true

Math nesnesi ile sayılar üzerinde işlemler yapabiliriz. Örnek olarak:

- Math.round(5.5) --> Sayıyı en yakın tam sayıya yuvarlar. Küsürat 5'ten küçün ise aşağıya, 5 ve 5'ten büyük ise yukarıya yuvarlar.
- Math.floor(5.3) --> Sayıyı aşağıya yuvarlar.
- Math.ceil(5.2) --> Sayısı yukarıya yuvarlar.
- Math.min(5,6,7,83,2,1) --> Verilen sayılar içerisindeki en küçük sayıyı döndürür.
- Math.max(6,83,88,34,99,1,223,55) --> Verilen sayılar içerisindeki en büyük sayıyı döndürür.
- Math.random() --> 0 ile 0.999999 arasında rastgele bir sayı üretir.
- Math.floor(Math.random() * 11) --> 0 ile 10 arasında rastgele bir sayı üretir.
- Math.abs(-10) --> Verilen sayının mutlak değerini alır.
- Math.sqrt(100) --> Verilen sayının karekökünü alır.
- Math.pow(3,2) --> Verilen ilk sayının, ikinci sayı kadar üssünü alır.
- Math.E --> Euler sayısını getirir. - Math.PI --> Pi sayısını getirir.

String birleştirmesi yapmak için farklı yollar vardır. '+' operatörü ile birleştirme yapmak kolay gibi görünse de hata yapmaya açık bir yöntemdir. Bunun yerine alternatif olarak şu yöneteme bakabiliriz, String Interpolation:

Bu birleştirme yöntemini bir string'in içerisine değişkenler ile değerler atamak istersek kullanabiliriz. Örnek olarak:
let a = 3
let b = 6
console.log(`${a} ile ${b} sayısının toplamı: ${a+b}'dir.`) --> 3 ile 6 sayısının toplamı: 9'dir.

JS'te her şey bir nesnedir.

Bazı String metotları:

- let str = 'JavaScript'
- str.length --> string'in karakter sayısını verir.
- str[0] --> str'nin ilk karakterini verir.
- str[str.length - 1] --> str'nin son karakterini verir.
- str.toUpperCase() --> str'nin bütün harflerini büyük harfe çevirir.
- str.toLowerCase() --> str'nin bütün harflerini küçük harfe çevirir.
- str.substr(2,3) --> str'nin 2. indexteki karakterinden başlayarak 3 adet ileri gider ve oluşan string'i döndürür.
- str.substring(2,3) --> str'nin 2. indexteki karakterinden başlayarak 3. index'e kadar gider.
- str.split(' ') --> str'yi, metot içerisine istenen karaktere göre parçalar ve parçaları bir array olarak geri döndürür.
- str.trim() --> str'nin başında ve sonundaki boşlukları (' ') temizler.
- str.includes('hede') --> Metoda verilen string değer, str içerisinde varsa true yoksa false döndürür.
- str.replace(old,new) --> str içerisindeki 'old' karakterleri 'new' karakterlere çevirir.
- str.charAt(0) --> str'nin istenen index'indeki elemanı döndürür.
- str.indexOf('hede') --> Verilen string ifadesi str içerisinde ilk kez kaçıncı index'te geçiyorsa o index'i döndürür. Eğer yok ise -1 döner.
- str.lastIndexOf('hede') --> Verilen string ifadesi str içerisinde en son kaçıncı index'te geçiyorsa o index'i döndürür. Eğer yok ise -1 döner.
- str.concat('hede','bidi') --> str'nin sonuna verilen string ifadeleri ekleyerek birleştirir.
- str.stratsWith('hede') --> str verilen string ile başlıyorsa true döner, başlamıyorsa false döner.
- str.endsWith('hede') --> str verilen string ile bitiyorsa true döner, bitmiyorsa false döner.
- str.search('hede') --> verilen string str içerisinde ilk defa nerede geçiyorsa o index'i döndürür. Parametre olarak regex pattern'i de verilebilir.
- str.match('hede') --> bir string veya regex pattern'i verilebilir. str içerisinde eşleşme olursa geriye bir array döndürür. Eğer eşleşme olmazsa null döner.
- str.repeat(4) --> str'yi verilen sayı kadar tekrar ettirir.

String to number:

- parseInt() --> parseInt('2')
- Number() --> Number('4')
- (+) --> +'10'

String to float:

- parseFloat() --> parseInt('9.81')
- Number() --> Number('9.81')
- (+) --> +'9.81'

Float to Int:

- parseInt() --> parseInt(9.81)
